# MeDoc Web-Referenz (`src/`)

Next.js-App (**nicht** das Tauri-Desktop-Produkt unter `../app/`).

- **Zweck:** Referenzimplementierung / Prototyp mit App Router, NextAuth, Prisma und PostgreSQL — z. B. für UI-Spikes oder eine spätere Web-Oberfläche.
- **CI:** Dieses Paket wird in derzeit **nicht** von `.github/workflows/ci.yml` gebaut; das kanonische Produkt liegt in **`../app/`**.
- **Build:** `npm ci && npm run build` (Next.js 16).

Verbindliche Traceability für das ausgelieferte Desktop-Produkt: **`../app/`** und **`../docs/v-model/00-uebersicht.md`**.
