# Figma exports for UI parity

Place exported frames here so `docs/ui/wireframe-route-map.md` can reference them by path.

**Naming:** `YYYY-MM-DD-<short-slug>.png` (example: `2026-04-25-dashboard-overview.png`).

**Suggested scale:** 2x PNG for legible type, or PDF for multi-screen flows.

The binary `.fig` file is not used directly in the repo; PNG/PDF (or a short written spec per screen) is enough for implementation reviews.
